### ソースコード
program112.c
```

#include <stdio.h>

int main(void){
    float x;

    for(x=0.0;x<=1.0;x+=0.01){
        #ifdef DBG
        printf("x = %f\n",x);
        #endif
    }
    printf("xの最後の値は%fです．\n",x);

    return 0;
}

```

### まとめ
- Cプリプロセッサはビルドでコンパイルの前段階で行われる．
- ライブラリファイルの読み込みやコメントの削除，マクロの置換などを行う．
- #ifdefや#endifは完成品には必要ないデバッグなどに使える．プログラムに残しておける
- 精度が必要な場合，float型は使わない．double型を必ず使う
- 繰り返しのカウンターは実数ではなく整数型を使う．
